# Recipes

Chaque recipe capture une solution fonctionnelle que l'on veut pouvoir rejouer.

## Format
- Contexte
- Decision
- Preconditions
- Commandes
- Validation

## Regle
- Si une solution fonctionne une fois et peut resservir, elle merite une recipe.
- Si une recipe est rejouee souvent, elle doit etre promue en script ou en target `Makefile`.
